Struttura del progetto:
/docs: contiene la traccia e la relazione finale
/gen: contiene i file generati da ANTLR per la grammatica di SimpLanPlus e SVM
/lib: contiene il .jar di ANTLR
/out: contiene l'output del programma
/src: contiene i sorgente del progetto, suddivisi in package

Per usare il programma dal relativo jar  utilizzare il comando:
    java -jar CLP_progetto.jar [path_to_input_file] [-help|-log|-skiptypecheck]
